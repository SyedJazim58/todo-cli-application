# Quickstart: Basic Todo CLI App

## Prerequisites
- Python 3.11 installed
- uv package manager installed

## Setup
1. Clone the repository
2. Navigate to the project directory
3. Install dependencies: `uv add`
4. Run the application: `uv run todo_cli`

## Usage
The application provides an interactive CLI interface with the following options:
1. Add Task: Create a new todo item with title and optional description
2. List Tasks: View all tasks with completion status indicators
3. Update Task: Modify task title or description by ID
4. Delete Task: Remove a task by ID
5. Mark Complete/Incomplete: Toggle task completion status by ID
6. Exit: Quit the application

## Example Workflow
1. Run `uv run todo_cli`
2. Select "Add Task" to create a new todo
3. Select "List Tasks" to view all todos with status
4. Select "Mark Complete" to update task status
5. Select "Exit" to quit the application
